.. _api-v1-user-schema:

============================
GET /api/v1/users/schema/
============================

Returns the current schema for the Consumer Notebook REST API ``user`` resource. 

Parameters
==========

access_token
    Oauth2 access token required for all Consumer Notebook REST API requests.

format (optional)
    * json (default)
    * jsonp

Example Request
================

Get::

    curl http://consumernotebook.com/api/v1/users/schema/ -d access_token={access_token} -G
